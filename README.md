# uC上面三个w的填空2025秒懂

　希望囡囡能够尽快恢复，走出妖怪的阴影。
　　不过，上次那个豹子肉味道真不错，自己的后院除了一个养不出鱼的池塘，并没有养什么家禽，想吃个荤腥挺不容易的。
　　而且，相比于普通的野味，妖怪的肉质显然会更好。
　　李念凡低声呢喃道：“倒是有些想吃野味了。”
　　他的声音不高，但白无尘等人是何许人也，听得一清二楚。
　　高人的暗示来了！
　　考验悟性的时候到了！
　　他们的眼眸都是大亮，激动得脸都红了。
　　“李公子，白某不才，愿意为您打些野味过来。”白无尘抢先开口说道。
　　“李公子，我也可以。”
　　林清云和赵山河落后一步，连忙异口同声道。
　　李念凡被他们的热情吓了一跳，“额……其实不用这么客气的。”
　　“李公子，我们免费听您的故事，给您打些野味是应该的。”林清云开口道。
　　白无尘已经迫不及待的御剑而起，“李公子不如先回去等待，我们这就去给您打野味。”
　　李念凡看他们这么积极，也不好佛了他们的好意，点头道：“如此就有劳了。”
　　话音刚落，白无尘等人便化为了长虹，一同飞了出去。
　　他们彼此互相看了一眼，眼眸中居然擦出了火花，显然是准备相互竞争，在李念凡面前表现自己。
　　就在这时，白落霜却是弱弱的开口道：“那个……我觉得李公子的话不会这么简单。”
　　嗯？
　　所有人都是一同看向白落霜。
　　林清云则是露出若有所思的神色。
　　“霜儿，你有什么想说的？”白无尘问道。
　　白落霜道：“爹，李公子肯定不是想吃野味这么简单，上次李公子灭了豹子精后，请我们吃了一顿烤豹肉，我觉得李公子所谓的野味应该是指妖怪。”
　　“我怎么没有想到？我刚刚注意到高人看囡囡的眼神中充满了怜爱，囡囡被妖怪害成这样，高人肯定会痛恨妖怪的！”钟秀连连点头。
